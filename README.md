# Cross-Window 3D Sync

Interactive 3D web experience that synchronizes across multiple browser windows in real-time using Three.js and WebGL.

## Features
- 🪟 Real-time window position tracking
- 🎨 Dynamic 3D particle system
- 🔗 Cross-window connection lines
- 💫 Responsive central sphere
- 🚀 GitHub Pages deployment

## How to Use
1. Open the deployed site in multiple browser windows
2. Move and resize the windows around your screen
3. Watch the 3D scene react in real-time!

## Tech Stack
- Three.js
- BroadcastChannel API
- Vite
- GitHub Actions
